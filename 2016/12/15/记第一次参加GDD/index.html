记第一次参加GDD

Posted on 2016-12-15 |

12月14号去上海参加了 google 开发者大会,这是我第一次参加开发者大会,头次参加开发者大会自然是很期待的.这次大会不同于其他国内的一些大会(介绍业内的开发经验),大会主要是介绍 google 的最新技术和开源项目.

刚开始的开场前预热是用Tilt Brush创作的3D 立体绘画表演,佩戴 VR 描绘上海 google 开发者大会的外景.接着 google 的产品总监,销售副总裁和 Angular 的项目负责人作了大会内容的简要介绍以及 对google 产品进行宣传.其中 google 的产品总监Andrew Bowers上台开始说了几句中文,以为他要用中文演讲,没想到他最后来了句他对他中文的评价是 “我对我的中文只能呵呵” ,接着开始用英文演讲hh.

接下去的几场都听了 Android 的,一些 Android 的演讲其实在 youtube 的 Android Developer 频道已经有发布了,只是做了部分的修改.

记录一下之前没有注意到到的功能:

  • 当 App 运行时发现内存不足的时候可以开启内存管理界面

    ACTION_MANAGE_STORAGE Added in API level 25
    Activity Action: Allows the user to manage their storage. This activity provides the ability to free up space on the device by deleting data such as apps.

  • 以及自带的输入法在高版本的设备中可以直接发送图片(不过聊天软件一般都做了).

  • Android开发中的电量和内存优化讲到了一些降低应用内存占用的方法和工具,其中Battery Historian可以让开发者检测电池在各阶段的使用信息,可以运行在 Api 版本为21的设备上.

下午的机器学习分会场参与的开发者很多, google 开源的 TensorFlow 和对深度学习的介绍很受欢迎.

听演讲的中间休息时间还玩了 DayDream,穿戴上装配了 Pixel 的 VR 眼镜来进行游戏,玩了个解谜类游戏,身处一个房间内部,当向左右看和仰视的时候设备通过重力感应为玩家呈现整个房间的布局. 听他们说目前对 Pixel 支持比较好, Nexus 的设备还有很多 bug.

GDD 这几年在中国举办表明了 google 对国内的开发者和市场十分重视,通过线下活动来帮助开发者使用google 的技术和推广他们的产品,参加这次开发者大会的体验还是蛮棒的,再一次充值信仰.
